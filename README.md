# vkCelShadingRenderEngine

## 简介

之前那个渲染器写的太烂，受不了了，重写

这个项目可以直接clone，然后在CLion里面运行。需要C++23（虽然我不确定是否用到了C++23的特性，但用了C++20，所以至少需要C++20）。
用了很多lambda表达式，可能不熟悉lambda的阅读会不太舒服。
